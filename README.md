🖐️ Hand Cursor Control

A Python-based hand gesture control system that allows users to control the mouse cursor using hand movements through a webcam.

---

📌 Features

- 🎥 Real-time hand tracking using webcam  
- 🖱 Control mouse cursor with hand movement  
- ⚡ Smooth and responsive tracking  
- 🤖 Computer vision powered interaction  

---

🛠 Tech Stack

- Python  
- OpenCV  
- MediaPipe  
- PyAutoGUI

🎮 How It Works

- The webcam detects hand landmarks in real time  
- Wrist movement controls the cursor
